# soa-exam-koa

Simple API written with Koa js for Service-Oriented Architectures exam

---

Examples of http requests can be found in [requests.rest](./requests.rest) file

## Run

Install dependencies
```
npm install
```

Start http server
```
npm run start:http
```

Start https server (require cert)
```
npm run start:https
```

Run tests
```
npm run test
```
